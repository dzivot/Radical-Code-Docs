About the Unified Stylesheet

All documents in this library share a single stylesheet: styles.css. This ensures consistent typography, color usage, spacing, and responsive behavior across all brand documentation.

Font Stack

  • Headlines: Zilla Slab Medium (tighter letter-spacing over 20px, wider when larger)
  • Subheads & Accents: Dosis Semibold/Medium
  • Body Text: Outfit Regular

Color System

All colours follow Brand Color Palette V3.1. See the dedicated colour palette document for complete guidance on gold system, section schemes, adjacency rules, and usage patterns.

Structure

Every page includes: consistent header with brand identity, "Back to Index" navigation, main content area with semantic structure, and footer with last-updated date.

Adding New Documents

When creating new brand documentation:

  • Link to styles.css in the head: <link rel="stylesheet" href="styles.css">
  • Use the standard header structure with brand row, eyebrow, h1, and description
  • Add a "Back to Index" link near the top
  • Use semantic HTML: <main>, <section>, etc.
  • Follow colour guidance from the palette document
  • Add a footer with document title and last-updated date
  • Update this index with a link to the new document

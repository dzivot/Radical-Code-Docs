Gold System — Dual-Surface Design

A single gold value cannot be optimised for both dark and light backgrounds simultaneously. The brand maintains two gold values, each designated for a specific surface context. This is standard practice in professional brand systems.

Gold Dark
Gold — Dark Surfaces
#f5b53d
Logo on dark azure, jade header/footer, jade accent sections
Gold Light
Gold — Light Surfaces
#e5a930
Logo on cream, warm honey, white backgrounds. Social assets.

Usage Rules

  • In site UI: Gold always means #f5b53d — the site operates primarily on dark surfaces (jade header/footer, dark azure sections) where Gold Dark is correct.
  • In social assets, print, and documents: Use Gold Light #e5a930 where the logo sits on cream or white backgrounds.
  • Accessibility: Never use gold as link or body text colour on cream backgrounds — contrast ratio fails WCAG AA. Use Jade Logo for links on cream.

Section Sequencing Rules

The jade header/footer architecture creates hard constraints that must be respected in every page layout. Violating these creates visual collisions that undermine palette coherence.

The Jade Adjacency Rule

Jade Shell (#018a75) and Jade Accent (#0B9C7E) are too close in value and hue to sit directly adjacent — they read as one muddy block rather than two distinct sections.

⚠ Critical: Jade Accent sections must never be placed directly below the header, directly above the footer, or adjacent to any other jade surface. Jade Accent sections must always be sandwiched between cream, dark cream, or dark azure surfaces.

Valid Adjacency Reference

Position Valid Schemes Notes
Header S4 Jade Shell Always the top surface. Brand shell announcement.
Entry section S3 or S1/S2 Must be dark azure OR cream. Never jade accent.
Interior sections S1/S2/S3/S5/S6 Any scheme except jade shell. Jade accent valid here.
Exit section S1/S2 or S3 Must be cream, dark cream, or dark azure. Never jade accent.
Footer S4 Jade Shell Always the bottom surface. Brand shell close.

Recommended Full-Page Rhythm

This sequence creates strong warm-clinical contrast throughout a full page:

Position Scheme Character
Header S4 Jade Shell Brand authority entry
Hero / First section S3 Dark Azure Immediate clinical weight. Dramatic open.
Content section A S1 Cream Light Living room warmth. Primary content surface.
Content section B S2 Cream Dark Rhythm alternation. Subtle differentiation.
Feature / Reverse S5 Jade Accent Brand energy moment. Interior only.
Content section C S1 Cream Light Return to warmth after jade accent.
Authority close S3 Dark Azure Clinical close. Lark bio, credentials, CTA.
Footer S4 Jade Shell Brand shell close.

Special Pattern: Secondary Pages

Pages without a hero image (blog posts, policy pages, secondary content) use jade page title treatment. The jade shell header flows directly into a jade page title section, creating unified brand entry before content begins.

Special Pattern: Blog Post Close

Each blog post closes with a Lark author bio section before the footer on S3 Dark Azure. The clinical authority reading is balanced by Lark's photo and personal copy, with a gold link as the action signal.
